## 📋 Project Overview
This repository contains the full source code for my personal responsive portfolio workspace. Shifting away from standard template architectures, the platform employs a bespoke dual-panel layout: a fixed left branding and identity node paired with a fluid, deep-scrolling right-hand viewport layout. It acts as an interactive, live-production resume designed to present complex technical projects, skills, and academic metrics clearly and attractively to recruiters, collaborators, and institutions.

---

## 💡 Key Design & Engineering Features
* **Bespoke Dual-Panel Layout:** Features a fixed, high-contrast left panel container providing persistent navigation and branding context alongside a clean, scrollable right content canvas.
* **High-Intensity Radiant Backdrop Glow:** Integrated hardware-accelerated ambient lighting logic behind the profile container that dynamically expands on user interaction loops.
* **Synchronized Interactive Telemetry:** Built with a clean typing engine for sub-headers and rhythmic, pulsing hyperlink indicators to cleanly direct user attention to live projects.
* **Fluid Mobile Bottom-Shelf UI:** Leverages refined responsive CSS media queries to morph complex desktop grid structures into an accessible, touch-optimized bottom navigation matrix for smaller viewports.
* **Defensive Form Interception:** Includes a fully custom client-side intake form equipped with strict structural verification constraints (regular expressions for emails and explicit character constraints for digits).

---

## 🛠️ Technology Stack & Architecture
* **Structural Blueprint:** HTML5 (Semantic Document Layout elements)
* **Design Token System:** CSS3 (Advanced Flexbox, Custom Root Variables, Component Keyframes, and Transitions)
* **Interaction Engine:** JavaScript (ES6+ Document Object Model API, Real-Time Active Section Scroll Tracking, and Hamburger Toggle States)
* **Production Deployment Infrastructure:** Vercel Global Edge Network & Cloud Architecture

## ⚙️ Local Deployment & Execution

To clone and run this project locally within a development environment:

1. Clone the Repository: git clone `https://github.com/suharshith2006-afk/portfolio.git`
2. Configure Local Environment: Move the project directory into your local XAMPP target folder: `C:\xampp\htdocs\portfolio`.
3. Launch the Engine: Open the XAMPP Control Panel, start the Apache module, and navigate to: http://localhost/portfolio/
